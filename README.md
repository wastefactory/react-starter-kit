# react-starter-kit
React starter kit with  webpack + babel

#Getting Started
There are two methods for getting started with this repo.

#Using Git
    > git clone git@github.com:StephenGrider/ReduxSimpleStarter.git
    > cd react-start-kit
    > npm install
    > npm start

#Downloading Zip
    > cd react-start-kit
    > npm install
    > npm start
